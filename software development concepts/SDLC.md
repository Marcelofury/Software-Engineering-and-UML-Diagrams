# Software Development Life Cycle (SDLC)

- SDLC is a structured process used to develop high-quality software efficiently.

## Stages:
- **Planning**: Identify problem, scope, feasibility
- **Requirement Analysis**: Gather user needs and system requirements
- **System Design**: Define architecture, database, UI
- **Implementation (Coding)**: Developers write the code
- **Testing**: Identify and fix bugs
- **Deployment**: Release system to users
- **Maintenance**: Updates, bug fixes, improvements
