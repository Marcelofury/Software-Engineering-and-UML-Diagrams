
# System Analysis vs System Design

| Aspect | System Analysis           | System Design                |
| ------ | ------------------------- | ---------------------------- |
| Focus  | What the system should do | How the system will do it    |
| Goal   | Understand requirements   | Create solution architecture |
| Output | Requirement documents     | Design diagrams (UML, ERD)   |
| Stage  | Early phase               | After analysis               |
