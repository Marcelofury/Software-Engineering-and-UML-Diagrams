#  Software Engineering & UML Diagrams Project

This repository contains UML diagrams and theoretical explanations for key Software Engineering concepts.

##  Contents

### 1. ATM System

* Activity Diagram for Cash Withdrawal
* Models user interaction with ATM including validation and transaction handling

### 2. Library System (OOP Design)

* Class Diagram showing:

  * Inheritance (User → Student, Lecturer, Librarian)
  * Book hierarchy (Book → ReferenceBook)
  * Associations between Users and Books
  * Abstract methods (e.g., `borrowBook()`)

### 3. Software Development Concepts

* Software Development Life Cycle (SDLC)
* Iterative vs Waterfall Development Models
* System Analysis vs System Design

### 4. Database Design

* ER Diagram for University Library System
* Entities: Book, Student, Staff, Loan
* Includes relationships and cardinalities
